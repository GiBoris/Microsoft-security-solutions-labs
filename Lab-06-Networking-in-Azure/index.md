# Azure Networking & Security Lab 🌐

This lab demonstrates the design and implementation of **secure Azure networking architecture**, including:

- Virtual Networks and subnet segmentation  
- Cross-network connectivity (peering & gateways)  
- Virtual WAN architecture  
- DDoS protection  
- Azure Firewall (application, network, DNAT rules)
